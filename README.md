# web3.skill

> 把 Web3 基础、币圈黑话、安全常识和开发入门，揉成一个真会说人话的 AI Skill。

一个偏中文语境的 `web3.skill`。  
它不是那种一开口就像百科词条的 Skill，而是更像一个混过几轮周期、踩过坑、也写过点东西的老 OG，在旁边直接把话说明白。

适合这几类问题：

- `什么是 web3？`
- `rug 是什么意思？`
- `CZ 是谁？`
- `币圈交易所有哪些？`
- `怎么开始接触链上？`
- `签名 / 授权安不安全？`
- `我想学 Web3 开发，从哪开始？`

## 它有什么不一样

- `先说人话，再讲术语`：不是一上来就一堆定义。
- `懂黑话，但不装神弄鬼`：`土狗`、`接盘`、`埋人`、`打狗`、`龙头` 这些都接得住。
- `像圈里人，但不喊单`：有判断，有味道，但不带你瞎冲。
- `安全边界清楚`：不会碰私钥、助记词，也不会把高风险操作说得像没事。
- `不只讲炒币`：钱包、链上交互、安全、DeFi、开发路径也都覆盖。

## 它会怎么说话

不是这种：

> Web3 是一种基于区块链、智能合约与去中心化网络构建的新型互联网范式。

而更像这种：

> 你先把 `web3` 理解成“拿钱包上网”。  
> 说白了，就是不只靠平台账号，而是用钱包、签名和链上规则跟应用打交道。

再比如：

**Q：`rug` 是什么意思？**  
**A：** `就是 rug pull，简单说就是项目方把桌布一抽，人全摔下去。圈里说某个币会 rug，基本就是在说它可能直接埋人。`

**Q：`接盘侠` 是什么意思？**  
**A：** `就是别人想卖的时候，你跑去高位把货接走了。圈里说你成接盘侠，基本就是你买在别人想出货的位置。`

**Q：`CZ` 是谁？**  
**A：** `一般就是赵长鹏，Binance 的创始人。圈里一提 CZ，默认基本都在说他。`

## 它能回答什么

### 1. Web3 基础

- 钱包、地址、私钥、助记词、签名
- Gas、公链、L1 / L2、稳定币、智能合约
- CEX 和 DEX 的区别

### 2. 币圈入门

- 常见交易所
- 现货、合约、杠杆、空投、meme、DeFi
- 新手怎么入场，哪些坑最容易踩

### 3. 炒币黑话

- `rug`
- `土狗`
- `接盘`
- `埋人`
- `拉盘 / 砸盘 / 洗盘`
- `上车 / 下车`
- `龙头 / 庄 / 主力 / 巨鲸 / 韭菜`

### 4. 安全常识

- 这个链接能不能点
- 这个签名能不能签
- 这个授权是不是有问题
- 常见钓鱼、假空投、假客服、无限授权风险

### 5. 开发路径

- Web3 开发先学什么
- EVM、RPC、ABI 是什么
- Solidity、Foundry、Hardhat、viem、wagmi 怎么串起来
- 怎么做第一个 dApp

## 安装

把这个仓库放到你的 Skill 目录里，并把文件夹名保持为 `web3`。

### Codex

如果你用的是 Codex，一般放这里：

- `$CODEX_HOME/skills/web3`
- 或 `~/.codex/skills/web3`

一个常见做法是直接 clone 到目标目录：

```bash
git clone https://github.com/watkin94/web3-skill.git ~/.codex/skills/web3
```

### Windows PowerShell

```powershell
git clone https://github.com/watkin94/web3-skill.git $HOME\.codex\skills\web3
```

### 其他 Agent

如果你用的是别的支持 `SKILL.md` 的 Agent，也可以直接复用这套内容。  
重点不是工具名，而是把整个目录作为一个 Skill 放到它能发现的位置。

## 这个 Skill 的边界

- 不提供个性化投资建议
- 不替你做买卖决策
- 不碰私钥、助记词、密码、API Key
- 不把高杠杆、盲签、无限授权说成“小问题”

简单说，它更像一个`会说人话的 Web3 / 币圈陪练`，不是一个`带单老师`。

## 适合谁

- 刚接触 Web3，想先听懂人话的人
- 能看懂一点英文，但更习惯中文黑话语境的人
- 想把 Web3 基础、币圈语境、安全常识一次打通的人
- 想从“会用钱包”走到“能做个小 dApp”的人
