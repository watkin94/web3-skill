# Pitfalls

Use this file when the user asks how people get rekt, what traps are common, or what a beginner should avoid.

The tone should be blunt but useful.
Do not moralize. Just show where people usually die.

## Top Beginner Killers

### Sending On The Wrong Network

- user copies the right address but chooses the wrong chain
- result can range from annoying recovery to permanent loss
- this is one of the most common beginner mistakes

### Blind Signing

- user sees "sign message" and assumes it is harmless
- result can be approval or permission they do not actually understand
- always tell the user to read the domain and the requested action

### Unlimited Approval

- user grants a contract too much spending power
- feels convenient in the moment, becomes dangerous later
- especially bad when the frontend or contract gets compromised

### High Leverage Too Early

- user thinks leverage is just a faster spot trade
- result is liquidation, emotional trading, and fast account damage
- if the user is brand new, default to "do not start here"

### Chasing Pumps

- user sees a chart already vertical and buys because of fear of missing out
- often becomes `接盘`
- explain that fast attention does not mean durable value

### Meme Coin Overexposure

- user treats `土狗` positions like core holdings
- result is often illiquidity, rug risk, or total collapse
- these should be framed as high-risk speculation, not beginner allocation

### One Wallet For Everything

- user uses the same wallet for experiments, airdrops, and meaningful holdings
- this increases blast radius if something goes wrong
- separate wallets by risk level

### Trusting Screenshots, KOLs, And Group Chat Certainty

- user copies a trade because somebody posted profit screenshots or loud conviction
- screenshots are not due diligence
- confidence in crypto is very cheap to manufacture

## Good Short Answers

- `新手最容易怎么死`
  - `不是没买到神币，是提错链、乱签名、开高杠杆、冲土狗。很多人不是输给认知不够，是死在最基本的操作错误。`

- `为什么大家老说别盲签`
  - `因为很多人以为“不是转账就没事”，结果签完才发现自己给了危险权限。这个坑特别适合埋新手。`

- `为什么别一上来就碰合约`
  - `因为你还没搞清楚波动、仓位和清算，杠杆只会把错误放大。对新人来说，这通常不是提速器，是绞肉机。`
