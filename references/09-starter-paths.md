# Starter Paths

Use this file when the user asks where to start.

Do not give the same path to everyone.
Route them into one of these three lanes.

## Lane 1: Pure Beginner

Use when the user barely knows the vocabulary.

### Goal

Understand the map before touching real money.

### Path

1. learn wallet, address, private key, seed phrase, and signature
2. learn what a chain is, what Gas is, and why networks matter
3. understand CEX vs DEX
4. use a fresh wallet and a tiny amount or a testnet

### First Warning

Do not rush from curiosity into leverage, meme coins, or random airdrop links.

## Lane 2: Crypto Newcomer

Use when the user already sees slang, exchange names, and project chatter, but lacks structure.

### Goal

Build market context without getting farmed by hype.

### Path

1. learn the core slang: rug, rekt, 接盘, 土狗, 龙头, 洗盘
2. understand the difference between spot, leverage, stablecoins, and on-chain usage
3. learn 5 recurring people and 5 recurring projects
4. learn the common ways beginners lose money

### First Warning

Do not confuse fluency in slang with actual understanding.

## Lane 3: Semi-Technical

Use when the user wants to understand the system deeply enough to build or debug things.

### Goal

Connect product usage, chain mental models, and developer tooling.

### Path

1. understand transactions, signatures, approvals, and RPC
2. learn Solidity basics and how contracts expose ABIs
3. build a tiny testnet app with wallet connect and one contract interaction
4. keep security thinking in the loop from day one

### First Warning

Do not hide behind code and ignore user-facing risk. In Web3, bad UX can be a security bug.

## Good Routing Examples

- `我完全不懂 Web3，从哪开始`
  - route to `Pure Beginner`

- `我天天刷到 rug、土狗、龙头，但看不懂`
  - route to `Crypto Newcomer`

- `我会前端，想学 Web3 开发`
  - route to `Semi-Technical`
